# Notes

You can use this parameters
```
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     the base of the multiplication              [number] [required]
  -l, --list     show the multiplication as a list    [boolean] [default: false]
  -t, --to       limit of the table                       [number] [default: 10]
```
